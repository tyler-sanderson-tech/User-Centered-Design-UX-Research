# User-Centered-Design-UX-Research

This repository documents a comprehensive **User-Centered Design (UCD) evaluation** of the **Me@Walmart** application, a critical enterprise-level tool [User Query]. The project showcases a complete UX research lifecycle—from initial planning and user interviews to the synthesis of actionable insights—designed to improve functionality and accessibility for a diverse workforce [User Query, 35, 146]. This work was completed as part of a **Bachelor of Science in Computer Science** at Wilmington University (Summa Cum Laude, 3.93 GPA).

## 🎨 Core Competencies & UX Methodologies
*   **UX Research Lifecycle:** Proficiency in developing research plans, crafting interview scripts, and conducting qualitative user interviews [User Query, 34, 35].
*   **User Modeling:** Creating detailed user personas and **customer journey maps** to visualize pain points and opportunities for improvement [User Query, 35].
*   **Accessibility Advocacy:** Hands-on experience with **screen reader accessibility simulations** to ensure inclusive design for all users [User Query, 35].
*   **Data Synthesis:** Ability to translate complex interview data into clear, high-level takeaways regarding design, navigation, and system performance.

## 📂 Repository Structure

### [Research-Phase](./Research-Phase)
The foundational stage of the UCD process, focusing on structured data collection:
*   **Refined Research Plan: Me@Walmart Project:** A strategic document outlining the goals and methodologies for the evaluation [User Query, 35].
*   **Me@Walmart Final Interview Script:** A standardized tool used to gather consistent qualitative feedback from users [User Query, 35].
*   **Me@Walmart User Interview Summary:** Analysis of raw interview data to identify recurring user themes and frustrations [User Query, 35].

### [Design-Analysis](./Design-Analysis)
Visualizing and testing the user experience through different lenses:
*   **Delivery App Customer Journey Mapping:** A detailed visualization of the user's end-to-end experience, highlighting emotional highs and technical lows [User Query, 35].
*   **Screen Reader Accessibility Simulation:** Documentation of accessibility testing results, emphasizing the importance of inclusive enterprise software [User Query, 35].

### [Final-Delivery](./Final-Delivery)
The culmination of the project, presenting findings and strategic recommendations:
*   **Final Presentation: Me@Walmart Insights:** A high-level report detailing what was learned about the app's design and navigation [User Query, 36].
*   **Key Takeaways:** Highlights from this presentation include insights on how improved UI can mitigate perceived performance issues and how to prioritize development based on user feedback.
